# BJJ video poster script for metricool

# Workflow

1. Start a chrome web browser in debug mode
2. Go to `https://app.metricool.com/planner`
3. Click the button with `aria-label="Create post"`
4. On the popup, click on the button that has an <i> tag under it with class="fa-regular fa-photo-video"
5. On the popup, click on the div with `aria-label="Add video"`
6. Select today's first video form `~/projects/carlson-gracie/content-poster/output`
7. After the video starts to upload, click on "Saved texts" button that has an <i> tag under it with class="fa-regular fa-folder-bookmark"
8. Select the first tag that has bjjdebrecen.hu in it
9. Set the date and time with the datepicker under div.mc-datetime-picker to today 19:00
10. Select facebook, instagram, tiktok, youtube icon at the top so that the video will be posted on those platforms
    - For facebook, select POST
    - For instagram, select REEL
    - For youtube, select SHORT
11. Say that you are ready
