# Claude Code Configuration                                                                                                                                                                                      
                                                                                                                                                                                                                   
Personal configuration and customizations for [Claude Code](https://claude.ai/code).                                                                                                                             
                                                                                                                                                                                                                   
## Contents                                                                                                                                                                                                      
                                                                                                                                                                                                                   
- **CLAUDE.md** — Global instructions applied to all projects                                                                                                                                                    
- **agents/** — Custom agent definitions                                                                                                                                                                         
- **commands/** — Custom slash commands                                                                                                                                                                          
- **hooks/** — Scripts that run on Claude Code events                                                                                                                                                            
- **output-styles/** — Custom output formatting styles                                                                                                                                                           
- **settings.json** — User preferences and permissions                                                                                                                                                           
- **skills/** — Custom skill definitions                                                                                                                                                                         
- **sounds/** — Notification sound files                                                                                                                                                                         
- **status_lines/** — Status line customizations                                                                                                                                                                 
                                                                                                                                                                                                                   
## Setup                                                                                                                                                                                                         
                                                                                                                                                                                                                   
Clone this repo to `~/.claude`:                                                                                                                                                                                  
                                                                                                                                                                                                                   
```bash                                                                                                                                                                                                          
git clone <repo-url> ~/.claude                                                                                                                                                                                   

Credentials, caches, session data, and plugins are not tracked — they are machine-specific or auto-managed by Claude Code.                                                                                       
```
